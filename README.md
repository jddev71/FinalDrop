# FinalDrop

**FinalDrop** es un videojuego del género *Battle Royale* diseñado con un enfoque en rendimiento y accesibilidad. El objetivo principal del proyecto es ofrecer una experiencia multijugador fluida que pueda ejecutarse en una amplia variedad de dispositivos, incluyendo aquellos con recursos limitados.

---

## Descripción

El proyecto integra sistemas modernos de desarrollo de videojuegos, incluyendo generación procedural de mapas y funcionalidades multijugador. Esto permite que cada partida sea única, al tiempo que fomenta la rejugabilidad y la interacción entre jugadores.

---

## Características principales

* Optimización para equipos de bajos recursos
* Generación procedural de escenarios
* Sistema multijugador en tiempo real
* Mecánicas inspiradas en el género *Battle Royale*

---

## Controles (provisionales)

* **W, A, S, D**: Movimiento
* **Mouse**: Control de cámara
* **Espacio**: Salto

---

## Tecnologías utilizadas

* Motor de juego: Godot
* Lenguaje de programación: GDScript

---
